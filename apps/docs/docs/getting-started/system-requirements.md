# System Requirements

## Devices

### For POS Terminals
- **Tablet** (recommended): 9th generation or newer, OS version 16+
- **Android tablet**: Chrome 90+
- **Desktop/laptop**: Any modern browser (Chrome, Firefox, Safari, Edge)

### For Kitchen Display
- Any tablet or monitor with a modern browser
- Recommended: wall-mounted tablet or touch screen

### For Managers
- Any device with a modern browser for reports and administration

## Network

- Stable internet connection for payment processing and delivery platform sync
- **Offline mode**: The POS works offline for cash orders — orders sync automatically when connection is restored
- Minimum recommended: 10 Mbps download / 5 Mbps upload

## Printers

- ESC/POS compatible thermal receipt printers
- Connection via network (recommended) or USB
- Supported widths: 58mm and 80mm

## Payment Hardware

- Stripe Terminal compatible card readers (for card payments)
- Or use manual card entry through the Stripe interface

## Browser Requirements

| Browser | Minimum Version |
|---------|----------------|
| Chrome | 90+ |
| Safari | 15+ |
| Firefox | 90+ |
