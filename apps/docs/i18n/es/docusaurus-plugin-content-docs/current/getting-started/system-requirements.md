# Requisitos del Sistema

## Dispositivos

### Para Terminales POS
- **Tablet** (recomendado): 9a generación o más reciente, versión de OS 16+
- **Tablet Android**: Chrome 90+
- **Escritorio/laptop**: Cualquier navegador moderno (Chrome, Firefox, Safari, Edge)

### Para Pantalla de Cocina
- Cualquier tablet o monitor con un navegador moderno
- Recomendado: tablet montada en pared o pantalla tactil

### Para Gerentes
- Cualquier dispositivo con un navegador moderno para reportes y administracion

## Red

- Conexion a internet estable para procesamiento de pagos y sincronizacion con plataformas de delivery
- **Modo offline**: El POS funciona sin conexion para pedidos en efectivo — los pedidos se sincronizan automaticamente cuando se restablece la conexion
- Minimo recomendado: 10 Mbps de descarga / 5 Mbps de subida

## Impresoras

- Impresoras termicas de tickets compatibles con ESC/POS
- Conexion via red (recomendado) o USB
- Anchos soportados: 58mm y 80mm

## Hardware de Pago

- Lectores de tarjeta compatibles con Stripe Terminal (para pagos con tarjeta)
- O usa la entrada manual de tarjeta a traves de la interfaz de Stripe
